    • Virtual Collections Prototype

    By using Dress X designs, this prototype exemplifies the basic function of the dress up game for the final site. Creating the prototype involved structuring appropriate files to operate using JavaScript. The final version will include recognisable brand collections for the metaverse and text pop-ups upon item selection (informing users about sustainable attributes).


    Step 1: creating html deciding on prototype components: background, base image (dressing stage), avatar, dress, and shoes.

    Step 2: CSS formatting and overlaying images: Images were screenshotted, removed of background, scaled to size of the avatar, modified using dimensions tool.

    Step 3: JavaScript file created to allow users to project different items onto avatar by clicking the next button.
